# Art Wall Builder

A premium Scandinavian-style gallery wall planner built as a React single-file JSX app.

Features:
- Multiple photo upload and drag-and-drop import
- Realistic textured wall backgrounds
- Draggable, resizable, rotatable framed photos
- Oak, dark oak, black, white, and custom frame materials
- Mat/passepartout toggle
- Layout presets
- Snap-to-grid
- PNG export via dynamically loaded html2canvas
- Save/load JSON layouts with embedded base64 images

## Run locally

```bash
npm install
npm run dev
```

Then open the Vite URL shown in the terminal.

## Build

```bash
npm run build
```
